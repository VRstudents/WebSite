Help and user guide

Registration

To access our platform, you need to sign up first. For now, we support registration with google account only.
Press "Sign Up" button in the upper right corner of the screen on any page, or one of the "Start ...." buttons on the home page.
When registering, choose your role - teacher or student and fill the relevant information. At this moment we have educational content only for mathematics and science, for the first grade only. That's why the grade selector will not allow you to set other grade, and the subjects selection is limited only for those two.

Navigation

From any page - click on the logo in the top middle of the screen to go to the home page.
If you're logged in, click on your initials in the upper right corner of the screen on any page, to go to your profile page, where you can see all your information and make all the necessary operations.

Teachers

As teacher, you can do several operations on the web site:

  • Open new class: from your profile page, click "Create new class" button, select a grade and a category. You can open a class only for one of the categories you've registered upon signing up.
    On the next screen you have to select which lessons you want to be available for the students in the class. Hovering on each icon will show you a description of the lesson. Mark the checkboxes near the icons to add the lessons to the class. Only added lessons will be available for the students in the game. You don't have to add all the lessons right away. You can add any of them at any moment later. In the end press "Create" button. You will see that the new class is added to the list of your classes.
  • Add lessons: from class page you can click on "Edit class" button. On the next page you will be able to add any lesson that is not already added to the class. Remember, you cannot remove already added lessons.
  • Create exam: Exams are custom made lessons, which can be created by you by choosing questions from a bank that we've created for you.
    On class page you can see if there is an exam already created for the class. You can create a new exam at any moment, but remember that creating a new exam will override the exam that may already exist for the class. Click on "Create exam" button". On the next page you will see a list of questions available for selection. You have to choose at least 8 and press "Save" to finish the process.
  • Post message to the class: Message that you post in the class will be visible for the students on the class page, and in the game. You can see the current class message that you've posted, and post a new one, which will override the previous one.
  • View statistics: On the class page you can view different statistics - for each lesson and each student in the class and overall class statistics.

Students

As a student you can play the lessons game. In order to access the games you must join a class - one for each category you want to play. To access the game you will need a personal access code. You can get the code from your profile page - press show code in your personal information section. The code is personal, so don't share it with others. The code is randomly regenerated each time you sign in into the website.

  • Join class: from your profile page, click "Join class" button and a list of available class from your school and grade will appear. You can see what is the subject of the class and who is the teacher. Select a class and click "Join class". You will see that the class has appeared in the list of your classes on the "Courses" tab on your profile page. Also you will see the available lessons for the newly joined class under "Classes" tab.
  • Play the games, sorry, the lessons: when you access the game, you can play all the lessons that are available for you in your classes. Result of the games are sent to the server and will be displayed for you in your personal statistics and for your teacher in the class statistics.
  • Take an exam challenge: In the game you can take a challenge exam. This is a custom exam that your teacher created for your class. Click on "Challenge yourself" i nthe app to get the available cahllenges.
  • View statistics: On your profile page, you can view your personal statistics under "Achievements" tab.
